HTML Lists

Lists are used to group together related pieces of information so they are clearly associated with each other and easy to read. In modern web development, lists are workhorse elements, frequently used for navigation as well as general content.

Types of Lists

  1. Unordered list — Used to create a list of related items, in no particular order.
  2. Unordered list — Used to create a list of related items, in a specific order.
  3. Description list — Used to create a list of terms and their descriptions.

HTML Unordered Lists

An unordered list is a collection of related items that have no special order or sequence. This list is created by using HTML <ul> tag, and each list item starts with the <li> element. The list items in unordered lists are marked with bullets.
